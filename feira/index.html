Componentes do PC

Processador

O processador é o cérebro do computador, responsável por executar instruções e processar dados.

Memória RAM

A memória RAM armazena temporariamente dados e instruções que o processador está usando.

Placa de Vídeo

A placa de vídeo é responsável pela renderização de gráficos e imagens.

Placa Mãe

A placa-mãe conecta todos os componentes do computador e permite que eles se comuniquem.

Fonte

A fonte fornece energia elétrica para todos os componentes do computador.

Placa de Áudio

A placa de áudio é responsável pela reprodução e captura de áudio no computador.

SSD

O SSD é uma unidade de armazenamento que usa memória flash para armazenar dados de forma rápida e eficiente.

HD

O HD é uma unidade de armazenamento tradicional que usa discos magnéticos para armazenar dados.

Cooler

O cooler é responsável por manter os componentes do computador, especialmente o processador, refrigerados.

Gabinete

O gabinete é a estrutura que abriga e protege todos os componentes do computador.

Placa de Rede

A placa de rede permite a conexão do computador a uma rede local ou à internet.
